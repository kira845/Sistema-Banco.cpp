# Sistema Bancario en C++

Proyecto desarrollado como práctica de programación estructurada en C++.  
Simula el funcionamiento básico de un sistema bancario con manejo de múltiples cuentas y persistencia de datos en archivos.

---

## Descripción

Este sistema permite la gestión básica de cuentas bancarias a través de una aplicación de consola.  
El programa implementa operaciones fundamentales como creación de cuentas, depósitos, retiros y almacenamiento de información en un archivo de texto.

El objetivo principal es reforzar conceptos de programación estructurada, modularización y manejo de archivos en C++.

---

## Funcionalidades

- Creación de cuentas bancarias
- Depósitos
- Retiros
- Búsqueda de cuentas por ID
- Listado de cuentas registradas
- Baja de cuentas
- Guardado de información en archivo `cuentas.txt`
- Carga automática de datos al iniciar el programa

---

## Tecnologías utilizadas

- C++
- Programación estructurada
- Manejo de archivos con `fstream`
- Estructuras
- Funciones
- Validación de entradas

---

g++ main.cpp -o banco
